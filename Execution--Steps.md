## Initiate uv
```bash
uv init .
```

## Install FastMcp

```bash
uv add fastmcp 
```

## Check FastMcp Version

```bash
fastmcp version
```

## Create a Server in main.py file -----

## Run Server

```bash
uv run main.py
```

## Inspect Your Server 

```bash
uv run fastmcp dev main.py
```

## Add MCP client

```bash
uv run fastmcp install claude-desktop main.py 
```
